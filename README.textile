h1. objc-generator 

The purpose of this project is to have a tool, that simplifies the daily work on the integration of XIB files, created using the Interface Builder, 
with the Controllers of the project views.

h2. External Libraries

CommandLine - Written by Ivo Kendra, 2009
nib2objc Classes Procesor - Written by Adrian Kosmaczewski 2009

h2. Example of Use

<pre>
objc-generator TestViewController.xib -c TestViewController -p exampleProject -b JuanCruzGhigliani -d 02/04/11 -n MyCompany -i UIViewController 
</pre>
